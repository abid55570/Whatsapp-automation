# Business — WhatsApp Auto for Bharat SMBs

## One-line
Affordable WhatsApp automation for Indian kirana, restaurant, salon, clinic, coaching — owner-only, mobile-first, 6 languages.

## Problem
- 95% Indian SMBs on WhatsApp. Owner = sole agent. Hands full.
- Same Qs every day: "open?", "menu?", "price?", "delivery hai?"
- Misses 30-40% inquiries after-hours. Lost sales.
- Existing tools (Wati, Interakt, AiSensy): ₹2,500–₹15,000/mo. Built for enterprise. English-first UI. Owner can't operate alone.

## Solution
- Auto-reply WhatsApp in customer's language (Hindi/Hinglish/English/Bengali/Urdu/Bhojpuri).
- Order parser: "do atta, ek dal" → cart → confirm → COD/UPI → pickup time.
- Google Sheet as CMS: owner edit sheet → bot updates menu/price.
- Owner sees inbox + orders on phone (PWA, installs like app, no Play Store needed).

## Target customer
- Tier-2/3 city kiranas, dhabas, salons, tutors, clinics.
- Owner age 25–55, smartphone-comfortable, broken-English ok.
- 50–500 WhatsApp inquiries/month.
- Currently: pen+paper + memory.

## Differentiation moats
1. **Sheet CMS** — owner edits Google Sheet; bot syncs. No "admin panel training".
2. **Hindi-script + Hinglish + Devanagari** — first-class. Competitors English-only.
3. **Pickup-first orders** — kirana don't deliver, they want pickup. We default to pickup w/ slot.
4. **Hindi-number parser** — "do atta, teen dal" → 2 atta + 3 dal. No competitor does this.
5. **₹399 floor** — 5–6× cheaper than incumbents. SMB-affordable.
